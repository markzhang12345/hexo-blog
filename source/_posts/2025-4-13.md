# devzat 简介

devzat 是程序员专属的 SSH 聊天室。这是一个通过 SSH 连接的聊天室，用户无需安装客户端，仅需一条 SSH 命令即可登录。它支持私人消息、多聊天室、图片和代码高亮等功能，还可以集成第三方服务、自托管 SSH 聊天室。

# 快速启动

## 环境要求

首先，确保你的电脑有 SSH 客户端。正常用过 git 的电脑中都应该有 SSH 客户端了，没有的可以去下载 OpenSSH

然后，确保你的电脑中存在 SSH 密钥对，虽然我并未设置对密钥进行认证，但是你得有一个

可以使用以下命令创建 SSH 密钥对：

- `ssh-keygen -t rsa -C "youremail@example.com"`
- `ssh-keygen -t ed25519 -C "your_email@example.com"`

基本上对密钥版本没有特殊的要求，选一个创建就行，输完命令之后一路回车即可

## 快速启动

打开任意终端，输入命令

````powershell
ssh <用户名>@60.205.131.158 -p 11451
````

即可连接至聊天室

有时候维护服务区重启项目，可能需要使用以下命令删去原先的指纹

```powershell
ssh-keygen -R "[60.205.131.158]:11451"
```

用户名就是你的网名，随便起就行

## 不快速的启动

每次输入一长串命令太麻烦？可以直接在用户文件夹下的`.ssh/config`中添加以下配置

```plainteXt
Host devchat          # 自定义别名（可随意命名）
    HostName 60.205.131.158
    Port 11451
    User <用户名>       # 你的用户名
```

**现在只需输入 `ssh devchat` 即可自动连接**
